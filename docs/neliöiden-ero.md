# Neliöiden ero

\(1^2 = 1\)

\(2^2 = 4\)

\(3^2 = 9\)

\(4^2 = 16\)

Kun tutkimme näitä lukuja, huomaamme, että neliöiden ero kasvaa aina kahdella

*Voimme todistaa tämän :*

\((n+1)^2 = n^2 + 2n + 1\)

Vähennetään alkuperäinen \(n^2\), jotta tiedemmä kuinka paljon luku kasvoi.

\(n^2 + 2n + 1 - n^2 = 2n + 1\)
